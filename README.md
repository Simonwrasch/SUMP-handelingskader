# SUMP Handelingskader

Een interactief instrument voor stedelijke knooppunten om hun positie in de mobiliteitstransitie te bepalen en gerichte adviezen te ontvangen.

Ontwikkeld als eindproduct van een afstudeeronderzoek aan Saxion Hogeschool, in opdracht van Witteveen+Bos.
